üìã Before You Start

Make sure you have:

  • Google Chrome browser installed
  • Downloaded the extension ZIP file

üö® Important Security Note

This extension requires manual installation because it's not available on the Chrome Web Store. Chrome will show security warnings - this is normal for manually installed extensions. The extension only accesses Facebook pages and operates locally on your computer.

üì¶ Step-by-Step Installation

1

Download & Extract Files

Download the extension ZIP file from the download page and extract it to a folder on your computer.

Recommended location: Desktop/facebook-messenger-exporter

What you should see:
  • manifest.json
  • popup.html
  • popup.js
  • USER_GUIDE.html
  • README.md
2

Open Chrome Extensions Page

Open Google Chrome and navigate to the extensions management page:

  • Method 1: Type chrome://extensions/ in the address bar
  • Method 2: Menu ‚Üí More Tools ‚Üí Extensions
  • Method 3: Press Ctrl+Shift+Delete then click Extensions
3

Enable Developer Mode

On the Extensions page, look for the "Developer mode" toggle in the top-right corner and turn it ON.

Don't see Developer mode?
  • Make sure you're on the Extensions page
  • Try refreshing the page
  • Check if you're signed into Chrome

After enabling, you'll see new buttons appear: "Load unpacked", "Pack extension", and "Update".

4

Load the Extension

Click the "Load unpacked" button that appeared after enabling Developer mode.

In the file browser dialog:

  1. Navigate to where you extracted the extension files
  2. Select the folder containing manifest.json
  3. Click "Select Folder" (or "Open" on Mac)
5

Verify Installation

If successful, you should see:

  • The extension appears in your extensions list
  • A download icon (üì•) appears in your Chrome toolbar
  • Extension status shows as "On"

üîí Security Information

Why Chrome Shows Security Warnings

Chrome displays warnings for all manually installed extensions as a security precaution. This extension:

  • Only accesses Facebook and Messenger pages
  • Processes data locally in your browser
  • Never sends data to external servers
  • Cannot access other websites or personal files

‚ö†Ô∏è Permissions Explained

  • Active Tab: Needed to read Facebook messages from the current tab
  • Downloads: Required to save exported messages to your computer
  • Storage: Stores your extension settings (like date format preferences)
  • Scripting: Allows the extension to interact with Facebook's webpage
